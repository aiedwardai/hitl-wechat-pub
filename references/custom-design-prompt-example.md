# custom 主题设计提示词示例

来源：2026-06-21 剑闻 | 马斯克万亿身家 · SpaceX涨37%
主题命名：**"星际富豪"**（深空×金色）

## 使用场景

当用户选择 `--theme custom` 时，md2wechat 返回的只是通用默认 AI 提示词，不包含主题设计意图。Agent 必须在 LLM 生成 HTML 时自行注入设计方向。

## 设计定位模板

```markdown
## 自定义主题偏好

### 设计定位
- 主题：**"[主题名]"**
- 色调：[主色] + [点缀色]
- 氛围：[1-2个关键词描述]

### 配色方案
- 主背景：`#XXXXXX`（[描述]）
- 卡片背景：`#XXXXXX`（[描述]）
- 主标题色：`#XXXXXX`（[描述]）
- 正文色：`#XXXXXX`（[描述]）
- 强调数字：`#XXXXXX`（[描述]）
- 分割线：`#XXXXXX`（[描述]）
- 引用/标签：`#XXXXXX`（[描述]）

### 排版规则
- [排版规则1]
- [排版规则2]
- ...
```

## 示例：星际富豪（深空×金色）

适合：马斯克万亿身家、航天/科技、金钱叙事主题

```markdown
### 设计定位
- 主题：**"星际富豪"**
- 色调：深蓝太空背景 + 金色数字点缀
- 氛围：科技感、金钱叙事、宏大叙事

### 配色方案
- 主背景：`#0B0E1A`（深空蓝黑）
- 卡片背景：`#13182A`（半透明蓝）
- 主标题色：`#E8C95A`（金色，万亿财富意象）
- 正文色：`#D4D8E0`（月白，易读）
- 强调数字：`#FFD700`（纯金，关键数据高亮）
- 分割线：`#1E2A4A`（星轨蓝）
- 引用/标签：`#2A3A6A`（星空蓝）
- 正/负数字：`#4CAF50`（绿）/ `#FF5252`（红）

### 排版规则
- 标题使用 emoji + 金色
- 关键数字加粗+金色高亮
- 每个板块用深色卡片包裹（`background: #13182A; border-radius: 10px; padding: 18px`）
- 来源链接用 ⭐ 代替 📎（星光符号，契合太空主题）
- 表格使用半透明深色底+金色表头
- 分割线使用渐变色：`linear-gradient(90deg, transparent, #1E2A4A, transparent)`
- 页面标题用 金色+字母间距 营造高端感
- 底部用星点装饰线 + 信息整合
```

## 通用 HTML 规范（所有主题通用）

```html
<!-- 外层容器：不用 <body> 设样式 -->
<section style="padding: 20px 15px; background: [主背景]; font-family: -apple-system, 'PingFang SC', 'Microsoft YaHei', sans-serif;">

  <!-- 每个段落必须带 color 防微信重置 -->
  <p style="color: [正文色]; font-size: 14px; line-height: 1.8;">

</section>
```

```
~/wechat-publish-{abbr}/design-prompt-{theme名}-{YYYY-MM-DD}.md
```
